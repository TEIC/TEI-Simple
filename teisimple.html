1. The TEI Simple schema

1.1. Summary

A total of 104 elements are selected for use in the <text> part of a document; an additional 40 elements are allowed for in the header. The following table shows the usage of all elements in six existing corpora, and classifies them 11 usage groups. There are 16 elements listed which are not allowed in TEI SImple, but should be transformed to another element.

1.3. The header

The default set of elements for the header are loaded using the header module. In addition, elements from other modules are loaded, if they are tagged in the classification as being needed for the header only.

Specification group [header]

Module: header

Elements which are only intended to be used in the header are banned from the <text>, using a Schematron rule.
<text>
[#headeronlyelement]
<constraint> <s:rule context="att| biblFull| biblScope| biblStruct| change| charDecl| charProp| editor| editorialDecl| email| encodingDesc| extent| fileDesc| gi| glyph| glyphName| idno| imprint| keywords| licence| listChange| listPerson| localName| monogr| msDesc| msIdentifier| physDesc| profileDesc| publicationStmt| relatedItem| repository| resp| respStmt| sourceDesc| teiHeader| textClass| textDesc| titleStmt| typeDesc| value">  <s:report test="ancestor::tei:text">Error: The element <s:name/>    is not permitted outside the header</s:report></s:rule></constraint>
